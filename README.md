# 🎓 AI Student Productivity Assistant

An AI-powered student productivity platform built to help students manage academic activities efficiently. The system integrates Generative AI, productivity tools, reminders, PDF summarization, study planning, and academic assistance into one centralized platform.

This project is developed to improve student productivity, simplify learning, and support academic management through intelligent assistance.

---

## 🚀 Features

### 🔐 User Authentication
- Secure Login System
- New User Registration
- Session-based Authentication

### 📘 PDF Assistant
- Upload study materials in PDF format
- Extract text from PDFs
- AI-powered PDF summarization
- Ask questions directly from uploaded notes

### 🧠 AI Chat Assistant
- Ask academic or general study-related questions
- Powered by **Google Gemini AI**
- **OpenAI fallback support** if Gemini quota exceeds

### 🗓️ AI Study Planner
- Generate personalized study plans
- Subject-wise scheduling
- Daily study hour planning

### ⏰ Reminder System
- Create assignment reminders
- Track deadlines and pending tasks

### ⌛ Time Manager
- Manage study hours
- Save productivity schedules
- Improve academic planning

### 📚 Learning Resources
- Store and manage educational resource links
- Subject-based resource management

### ☁️ Drive Storage Simulation
- Upload and manage study documents
- Store uploaded filenames in database

### 📧 Email Reminder System
- Save email reminders
- Notification management for students

### 🛠️ Admin Panel
- Dashboard analytics
- User statistics
- Module usage tracking
- System monitoring

## 💡 How the System Works

1. Users log in or register into the system.  
2. Students can upload PDFs for summarization and question answering.  
3. AI chat provides instant academic assistance.  
4. Study plans are generated based on subjects and study hours.  
5. Reminders help students track assignments and deadlines.  
6. Time management tools improve productivity.  
7. Learning resources can be stored for quick access.  
8. Admin panel monitors analytics and system activity.
